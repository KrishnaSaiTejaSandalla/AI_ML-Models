# Student Performance Prediction using Machine Learning

## Problem
This project predicts whether a student will pass or fail
based on study hours, attendance, and previous scores.

## Features
- Study Hours
- Attendance
- Previous Score

## Model Used
Logistic Regression

## Evaluation
Used train-test split.
Measured performance using accuracy and confusion matrix.

## How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Train model:
   python train.py

3. Predict:
   python predict.py
